# BDW-BeyondMusic 🎶  

Ein moderner, portabler Musik-Player für Windows – entwickelt mit **Python**, **PySide6** und **VLC**.  

---

## ✨ Features  
- 🎵 Unterstützung für **MP3, FLAC, WAV** und mehr (via `mutagen` + `vlc`)  
- 🎨 Moderne Benutzeroberfläche mit **PySide6**  
- 📂 Playlist-Management & einfacher Dateiimport  
- 🖼️ Anzeige von Albumcovern (ID3-Tags)  
- ⚡ Portable & leichtgewichtig  

---

## 🛠️ Installation (Source Version)  

### Voraussetzungen  
- **Python 3.10+**  
- Abhängigkeiten aus `requirements.txt`:  
  ```bash
  pip install -r requirements.txt
````

### Starten

```bash
python app.py
```
